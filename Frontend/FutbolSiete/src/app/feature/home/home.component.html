Bienvenido a Futbol Siete

Reserva ya tu cancha de preferencia para jugar con tus amigos

Aquí podrás tener a disposición los siguentes servicios:


Cliente
Crear las credenciales

Se puede crear un usuario con sus datos personales, para poder realizar una reserva de la cancha.

Cliente
Eliminar las credenciales

Si el cliente desea no pertenecer más a la tienda, se podrá eliminar su usuario cuando deseé.

Cancha
Agregar cancha

Se puede crear una cancha nueva para estar a disposición de ser reservada inmediatamente, siempre y cuando cumpla las reglas de negocio.

Cancha
Reglas de negocio

No se puede crear una reserva los lunes, se hace mantenimiento este día, se hacen descuento dependiendo la fecha que reserves.

Reserva
Reservar una cancha

Podrás reservar cualquier cancha, a la hora que desees, siempre y cuando tengas un usuario creado y exista la disponibilidad.
